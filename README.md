# Лабораторная работа: Система контроля версий Git

## Описание
Данный репозиторий содержит материалы лабораторной работы, посвящённой изучению системы контроля версий Git, работе с ветками, коммитами и разрешению конфликтов. Работа предполагает создание веток, внесение изменений в файлы, сравнение коммитов и безопасное слияние изменений.

## Структура репозитория
- `config.txt` — основной файл, используемый для внесения изменений в рамках практической работы.  
- `diff.txt` — файл с сохранённым выводом разницы между коммитами.  
- `history.txt` — файл с историей коммитов.  
- `README.md` — данный файл с описанием лабораторной работы.
